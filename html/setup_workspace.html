Allow Hiera Editor to retrieve facts from another nodes.
To do so, you would need to edit file auth.conf on the Puppet Server:
nano /etc/puppetlabs/puppetserver/conf.d/auth.conf

Scroll down (with arrow keys) until you see a block like this
# Allow nodes to retrieve only their own node definition
match-request: {
    path: "^/puppet/v3/node/([^/]+)$"
    type: regex
    method: get
}
allow: ["$1", ]
sort-order: 500
name: "puppetlabs node"
Update that block as shown in the red line above (allow: [...]).
